### WeMediaForWordPress付费阅读插件
---

本插件可以隐藏文章中的任意部分内容，当访客付费后，可查看隐藏内容，当前版本仅支持SPay微信支付，而且SPay支付偶尔会回调服务器时间过于慢，影响业务处理，后期会增加其他支付，请知悉。

#### 使用方法：
第一步：下载本插件，放在 `wp-content/plugins/` 目录中（插件文件夹名必须为WeMedia）；

第二步：激活插件；

第三步：填写配置；

第四步：完成。

#### 版本说明：
此插件使用php5.6+Wordpress5.0.2编写
